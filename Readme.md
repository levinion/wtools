## wtools

wtools包含一些适合于Windows系统日常工作的脚本，并可配合Powertoys提供的keyboardmanager工具实现类似i3的效果：



- 窗口（wtools window）

| 关闭窗口         | close               |
| ---------------- | ------------------- |
| 最小化窗口       | minimize            |
| 最大化窗口       | maximize            |
| 还原窗口         | restore             |
| 最大化或还原窗口 | maximize-or-restore |

- 桌面 (wtools desktop)

| 左移一个桌面                    | switch-left  |
| ------------------------------- | ------------ |
| 右移一个桌面                    | switch-right |
| 移动至 \<Index\> (Start from 1) | switch-to    |

- 系统 (wtools system)

| 锁定 | lock |
| ---- | ---- |
